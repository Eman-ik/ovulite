    - Install: tailwindcss, postcss, autoprefixer

    - Configure tailwind content paths: `["./index.html", "./src/**/*.{js,ts,jsx,tsx}"]`

    - react-router-dom (v6) for routing
    - axios for API calls
    - clsx + tailwind-merge for className utility (shadcn pattern)
    - lucide-react for icons

    Create `frontend/src/lib/utils.ts` with the standard shadcn `cn()` utility:
    ```typescript
    import { clsx, type ClassValue } from "clsx";
    import { twMerge } from "tailwind-merge";
    export function cn(...inputs: ClassValue[]) { return twMerge(clsx(inputs)); }
    ```

    Add a few base shadcn/ui components manually (button, input, card, label) — minimal versions following shadcn patterns. These will be used by the login page.

    Configure vite.config.ts with path alias: `@` → `./src`

    **Do NOT install all 50+ shadcn/ui components.** Only button, input, card, label for now. More will be added as needed.
