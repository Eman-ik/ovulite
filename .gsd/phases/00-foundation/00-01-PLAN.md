<task type="auto">
  <name>Task 2: Dockerfiles + docker-compose.yml</name>
  <files>
    backend/Dockerfile
    frontend/Dockerfile
    docker-compose.yml
  </files>
  <action>
    Create `backend/Dockerfile`:
    - FROM python:3.11-slim
    - WORKDIR /app
    - COPY requirements.txt and pip install
    - COPY app/ into /app/app/
    - CMD ["uvicorn", "app.main:app", "--host", "0.0.0.0", "--port", "8000", "--reload"]
    - EXPOSE 8000

    Create `frontend/Dockerfile`:
    - FROM node:20-alpine
    - WORKDIR /app
    - COPY package.json package-lock.json* ./
    - RUN npm install (will work once package.json exists from Plan 03)
    - COPY . .
    - CMD ["npm", "run", "dev", "--", "--host", "0.0.0.0"]
    - EXPOSE 5173

    Create `docker-compose.yml`:
    ```yaml
    version: "3.9"
    services:
      db:
        image: postgres:15-alpine
        env_file: .env
        volumes:
          - pgdata:/var/lib/postgresql/data
        ports:
          - "5432:5432"
        healthcheck:
          test: ["CMD-SHELL", "pg_isready -U $POSTGRES_USER -d $POSTGRES_DB"]
          interval: 5s
          timeout: 5s
          retries: 5

      backend:
        build: ./backend
        env_file: .env
        ports:
          - "8000:8000"
        volumes:
          - ./backend:/app
          - ./ml:/app/ml
          - uploads:/app/uploads
        depends_on:
          db:
            condition: service_healthy

      frontend:
        build: ./frontend
        ports:
          - "5173:5173"
        volumes:
          - ./frontend:/app
          - /app/node_modules
        environment:
          - VITE_API_URL=http://localhost:8000
        depends_on:
          - backend

    volumes:
      pgdata:
      uploads:
    ```

    Create a minimal `backend/app/main.py` with FastAPI app + health check so the backend container can start:
    ```python
    from fastapi import FastAPI
    from fastapi.middleware.cors import CORSMiddleware

    app = FastAPI(title="Ovulite API", version="0.1.0")

    app.add_middleware(
        CORSMiddleware,
        allow_origins=["http://localhost:5173"],
        allow_credentials=True,
        allow_methods=["*"],
        allow_headers=["*"],
    )

    @app.get("/health")
    async def health_check():
        return {"status": "healthy", "service": "ovulite-api"}
    ```

    **Important:** Do NOT use `&&` in Dockerfiles CMD — use JSON array form. Ensure volumes mount correctly for hot-reload in development.
  </action>
